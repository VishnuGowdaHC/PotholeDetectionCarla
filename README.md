# Pothole Detection Carla

## About CARLA Simulator

> **CARLA** (Car Learning to Act) is an open-source simulator for autonomous driving research. It provides a flexible platform for developing, training, and validating autonomous vehicle algorithms in a controlled, repeatable environment.

### Key Features
- Realistic urban environments and traffic scenarios
- Physics-based vehicle dynamics
- Multi-sensor support (cameras, LiDAR, radar)
- Python API for easy integration
- Support for reinforcement learning and imitation learning
- Cross-platform compatibility (Linux, Windows, macOS)
